# Haxe Electron Boilerplate

![Haxe Electron Image](https://i.imgur.com/QwhIIF3.png)

## Commands

```sh
haxelib install electron

haxe build.hxml

yarn && yarn start
```

## Tecnologies

[Haxe](https://haxe.org/)

[Electron](https://www.electronjs.org/)

[HaxeElectron](https://github.com/tong/hxelectron)
